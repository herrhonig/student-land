## 🚀 How to run project

### 1. Install dependencies

```bash
pnpm install
```

### 2. Environment setup dependencies

```bash
VITE_API_URL=
VITE_API_REQUEST_DELAY=
```

### 3. Run app

```bash
pnpm run dev:all
```
